    Zensical - Installation

    Installation de Zensical sur le serveur S51 supportant déjà Mkdocs.

    Création zone de travail

    Créer le dossier suivant :

    mkdir /home/userx/Documents/notes-zensical

    Création environnement Python

    Créer l'environnement virtuel Python comme suit :

    cd /home/userx/Documents/notes-zensical
python3 -m venv .venv

    L'environnement est créé dans le dossier /home/userx/Documents/notes-zensical/.venv/.

    Activer celui-ci :

    source .venv/bin/activate

    Pour quitter l'environnement virtuel :

    deactivate

    Installation de zensical

    Il est nécessaire que l'environnement Python soit activé, ensuite.

    pip install zensical

    La structure suivante est créée :

    .
    ├─ .github/workflows
    │ └─ docs.yml
    ├─ docs/
    │ ├─ index.md
    │ └─ markdown.md
    └─ zensical.toml

    Mise à jour

    pip show zensical
pip install --upgrade --force-reinstall zensical
pip show zensical

    Création du site Web

    zensical new .

    Un dossier nommé par défaut site est créé, faire une copie de celui-ci sous par exemple le nom public :

    cp -r site public

    Configuration

    Editer le fichier zensical.toml :

    nano zensical.toml

    et le modifier par exemple comme suit :

    prévisualisation du site Web

    Lancer le serveur local de prévisualisation :

    zensical serve

    et utiliser l'URL suivante :
    http://localhost:8000

    Pour fermer le serveur de prévisualisation, utiliser la combinaison de touche CTRL+c.

    Site Web de production

    Créer le script suivant :

    nano build.sh

    et entrer le contenu suivant :

    #!/bin/bash

zensical build &&
sudo rsync -av --delete public/ /var/www/html/zensical/

    Rendre le fichier exécutable :

    chmod +x build.sh

    Créer ensuite dans /var/www/html/ le dossier zensical

    sudo mkdir /var/www/html/zensical

    et créer le site web statique comme suit :

    ./build.sh

    Le site doit maintenant être joignable en utilisant l'URL indiquée dans le fichier zensical.toml soit :

    https://nom-de-domaine:port/zensical/

    Fin.
